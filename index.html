3 Core Automation Strategies
시급한 3대 영역 자동화 전략
고객관리 · 원가/예산 · 고객문서관리 — 실질적 자동화 방안
①
고객 커뮤니케이션 자동화
납기 문의·이메일·클레임 대응
⚠ 현재 고통
수십 개 프로젝트 고객에서 동시에 납기 문의·도서 검토 결과·변경 요청이 쏟아짐. 담당자가 이메일만 처리하다 하루가 끝남. 미답변 누락으로 고객 불만 발생.
✅ 자동화 솔루션
고객 이메일 수신 → 프로젝트 번호 자동 인식 → 유형 자동 분류 (납기문의/도서검토/변경/클레임)
미답변 48시간 초과 시 담당자+팀장 Teams 자동 알림
Copilot으로 납기 변경 통보 이메일 초안 자동 생성 (한/영)
고객별 커뮤니케이션 이력 Power BI 자동 대시보드
납기 D-30/14/7 도달 시 고객 사전 통보 이메일 자동 발송
Power Automate M365 Copilot Power BI
이메일 처리 60%↓ 누락 0건
②
원가·예산 실시간 자동화
조기경보·Forecasting·월말 집계
⚠ 현재 고통
SAP에서 데이터 뽑아 Excel 가공 반복. 월말이 돼야 원가 초과 파악. Forecasting 업데이트 수작업. 프로젝트 수 증가로 집계 시간 폭증.
✅ 자동화 솔루션
SAP ↔ Power BI 실시간 직접 연결 → Excel 수동 작업 완전 제거
원가 소진율 > 진도율 +10% 시 자동 경보 (Yellow), +20% 시 팀장 자동 보고 (Red)
EAC 자동 계산 공식 내장 → Forecasting 수작업 70% 제거
Power BI Copilot: "이번 달 원가 초과 프로젝트" 자연어 조회
SAP 버전 S/4HANA 확인 시 → Joule로 자연어 ERP 직접 조작
SAP↔Power BI 연동 Power Automate SAP Joule (조건부)
월말 집계 75%↓ 조기경보 자동화
③
고객 문서(서브미탈) 자동화
DCL 추적·알림·검토결과 관리
⚠ 현재 고통
수백~수천 건 서브미탈을 Excel DCL로 수동 추적. 자동 알림 없어 담당자가 매일 Excel 열어 확인. 기한 누락 시 계약 위반·클레임 발생.
✅ 자동화 솔루션
Excel DCL → SharePoint List 전환 → 자동화 기반 구축
제출 기한 D-14/7/1 자동 알림 → Teams + 이메일
기한 초과 시 팀장+PM 즉시 자동 에스컬레이션
고객 검토 결과 이메일 수신 → 상태 자동 업데이트 (Approved/RFA/Rejected)
Rejected 시 재제출 담당자 자동 배정 + 기한 자동 설정
전체 서브미탈 현황 Power BI 대시보드 자동 갱신
SharePoint + Lists Power Automate Power BI
누락 리스크 Zero 추적 업무 80%↓
🔧 추가 자동화: PR/PO 처리 자동화 (구매 폭증 대응)
표준 자재(반복 구매) → SAP iRPA 또는 SAP Workflow로 사람 개입 없이 자동 PO 생성. 고단가·신규·특수 자재만 사람이 검토하는 구조로 전환하면 PR/PO 처리 건수의 60~70%가 자동화됩니다. SAP iRPA SAP Workflow
