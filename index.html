🏢 핵심 기능 & 아키텍처
  • Joule Copilot: SAP 전체 모듈 자연어 조작 — "이번 달 원가 초과 프로젝트 TOP5" 즉시 분석
  • Joule Agents (2024 출시): PR 생성 → 공급업체 선정 → PO 발행을 사람 없이 자율 처리
  • SAP Business AI: FI(재무)/CO(원가)/MM(구매)/SD(영업)/PP(생산) 전 모듈 AI 내장
  • SAP Analytics Cloud 연동: 프로젝트 원가·수익 예측 AI 대시보드 자동 생성
  • 데이터 보안: SAP Trust Center 내부 처리 — 외부 전송 없음 (보안 민감 산업 최적)
⚡ 두산에너빌리티 적용 6대 Use Case
  • 계약 모니터링: 계약 마일스톤 자동 감지 → 청구 문서 자동 생성
  • 구매 자율화: 장납기 자재(터빈 블레이드 등) 재주문 자동 실행
  • 원가 조기경보: WBS별 원가 이상 자동 감지 및 담당자 알림
  • 품질 관리: 입고 검사 자동 생성, 공급업체 품질 점수 자동 갱신
  • 진도 청구: 마일스톤 달성 시 청구 문서 자동 생성 (IFRS 15 준수)
  • 현금흐름 예측: 수주 잔고 기반 AI 현금흐름 예측 자동 생성

🏢 핵심 기능 — SAP PS가 못 하는 것을 EcoSys가 한다
  • EVM (획득가치관리): BCWS/BCWP/ACWP, CPI/SPI/TCPI 자동 계산 — ANSI/EIA-748 완전 준수
  • 다중 예측법: CPI 기반, SPI 기반, 복합, 관리자 EAC 동시 운영
  • 변경관리 전 주기: 잠재 변경(Trend) → 승인 → 원가·일정 통합 반영
  • 포트폴리오 관리: 전체 프로젝트 포트폴리오 대시보드, 리스크 점수, 게이트 관리
  • SAP S/4HANA 양방향 통합: SAP CO/PS 실제원가 → EcoSys 자동 반영; EcoSys 예산 → SAP PS 자동 업데이트
  • 현금흐름 예측: 시간대별 원가·수익 예측으로 자금 계획 지원
⚡ 두산에너빌리티 핵심 적용 시나리오
  • 발전기기 제조 EVM: 설계완료→자재입고→제작완료→FAT→출하 마일스톤 기반 진도율 자동 계산
  • 다년도 프로젝트 원가 전망: 2~5년짜리 터빈/보일러 프로젝트 연간 원가 전망 자동 갱신
  • 변경계약 영향 즉시 분석: 고객 스코프 변경 시 원가·납기 영향 즉시 시뮬레이션
  • 임원 포트폴리오 뷰: 전체 수주잔고 프로젝트 원가 성과 한눈에 파악
